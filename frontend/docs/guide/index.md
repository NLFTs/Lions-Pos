# Pengenalan

Selamat datang di dokumentasi **Gaptek**. Dokumentasi ini dirancang untuk memudahkan pengembang dan pengguna memahami sistem secara menyeluruh.

## Tentang Gaptek

Gaptek adalah platform manajemen bisnis terintegrasi yang mencakup:

1. **Dashboard & Analitik** — Pantau data, tren penjualan, dan performa bisnis secara *real-time*.
2. **Manajemen Inventaris** — Kelola produk, stok, mutasi stok, gudang, dan cabang.
3. **Point of Sale (POS)** — Aplikasi kasir responsif untuk mencatat transaksi penjualan.
4. **Audit Logs** — Riwayat aktivitas HTTP pengguna untuk audit dan keamanan.

## Persyaratan

Sebelum memulai, pastikan:

- **Hak Akses** — Akun Anda memiliki *permissions* yang sesuai.
- **Cabang/Gudang** — Beberapa fitur memerlukan pemilihan lokasi aktif sebelum transaksi.
