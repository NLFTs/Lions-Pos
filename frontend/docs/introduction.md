# Pengenalan

Selamat datang di Dokumentasi Sistem **Gaptek**. Dokumentasi ini dirancang dengan antarmuka yang sangat minimalis untuk memudahkan pencarian informasi secara cepat dan efisien.

## Fitur Utama

Sistem ini memiliki beberapa komponen penting yang saling terintegrasi:

1. **Dashboard & Analitik**: Memantau ringkasan data, tren penjualan, dan performa bisnis secara *real-time*.
2. **Manajemen Inventaris**: Pengelolaan produk, stok saldo, mutasi stok, gudang, dan cabang.
3. **Point of Sale (POS)**: Aplikasi kasir responsif untuk mencatat transaksi penjualan.
4. **Audit Logs**: Riwayat aktivitas HTTP pengguna untuk tujuan audit dan keamanan.

## Persyaratan Awal

Sebelum memulai pengembangan atau penggunaan sistem, pastikan Anda memahami beberapa hal berikut:
- **Hak Akses**: Pastikan akun Anda memiliki *permissions* yang sesuai.
- **Cabang/Gudang**: Beberapa fitur memerlukan Anda untuk memilih lokasi aktif sebelum melakukan transaksi.

::: tip Info Cepat
Gunakan fitur pencarian di pojok kanan atas halaman (`Ctrl + K` / `Cmd + K`) untuk mencari dokumen atau topik tertentu secara langsung.
:::
