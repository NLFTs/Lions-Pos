# Showcase

Modul **Showcase** memungkinkan Anda menampilkan katalog produk kepada pelanggan melalui halaman etalase publik yang menarik.

## Fitur Utama

- **Halaman Etalase Publik** — Tampilkan produk tanpa perlu login.
- **Filter & Pencarian** — Pelanggan dapat menyaring produk berdasarkan kategori atau harga.
- **Mode Berbagi** — Bagikan tautan etalase langsung ke media sosial atau WhatsApp.
- **Branding Kustom** — Sesuaikan nama toko, logo, dan warna tema etalase.

## Mengaktifkan Showcase

1. Masuk ke menu **Pengaturan → Showcase**.
2. Aktifkan toggle **Tampilkan Etalase Publik**.
3. Pilih produk yang ingin ditampilkan (semua atau tertentu saja).
4. Salin URL etalase dan bagikan ke pelanggan.

::: info
URL etalase mengikuti format: `https://yourdomain.com/showcase/{slug-toko}`
:::

## Kustomisasi Tampilan

Anda dapat mengatur tampilan etalase melalui:

| Pengaturan       | Keterangan                                  |
|------------------|---------------------------------------------|
| Nama Toko        | Nama yang tampil di halaman etalase         |
| Logo             | Upload logo toko (PNG/JPG, maks 2MB)        |
| Warna Tema       | Pilih warna utama sesuai identitas brand    |
| Deskripsi Toko   | Teks singkat tentang toko Anda              |

## Integrasi WhatsApp

Aktifkan tombol **Pesan via WhatsApp** agar pelanggan dapat langsung menghubungi Anda:

```
Pengaturan → Showcase → Integrasi → Nomor WhatsApp
```

Masukkan nomor dalam format internasional tanpa tanda `+`, contoh: `628123456789`.
